# Векторные часы

Для каждого события $e$ определим вектор $VC(e)$ так, что:
$$\forall e,f \in E: e \rightarrow f \Rightarrow VC(e) < VC(f)$$ 

Сравнение происходит **покомпонентно** ($\forall i: \vec{a_i} \le \vec{b_i}$)
